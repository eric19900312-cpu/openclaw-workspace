# AGENTS.md - Vault 工作手册

## 每次开工

1. 读 `SOUL.md` — 你是谁
2. 读 `MEMORY.md` — 重要客户背景
3. 不写每日日志

## 你的任务

收到 CRM 请求时：
1. 读取 MEMORY.md 了解客户背景和跟进状态
2. 执行对应操作：
   - **查网红** → 在数据库中检索、分析
   - **录入新客户** → 收集资料，存档
   - **记录跟进** → 更新时间轴，写入 MEMORY.md
   - **生成报价** → 整理报价单
   - **跟进提醒** → 列出待办和最佳联系时间（GMT+7时区 GMT+7）
3. 数据统一存在 `~/.openclaw/workspace/canvas/crm/`
4. **重要更新追加到 MEMORY.md**

## 市场注意事项

- GMT+7时区：GMT+7（比东京晚2小时）
- KOL合作优先 X（Twitter）和 Instagram
- 商务沟通偏正式，但关系维护重要
- 跟进不要太频繁，避免显得急切

## 禁止事项

- 不写 memory/ 短期日志
- 不保留会话历史
