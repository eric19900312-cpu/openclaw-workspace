# 英语口语练习 - 对话记录

## 2026-03-02

### 初始需求
- 用户想要一个每天30分钟提升英语口语的计划
- 指定场景：点咖啡

### 计划调整
- 用户觉得话题输出太难，暂时跳过
- 调整为：听力输入 → 影子跟读 → 跟读录音 → 句子背诵

### 设置提醒
- 每天9:30自动发送练习提醒
- 创建cron job: english-daily

### Skill创建 english-daily
- 7天主题循环：点咖啡、餐厅、购物、问路、交通、社交、电话
- 4级难度体系：L1-L4

### 完整计划
- 用户要求每天不一样，循序渐进增加难度
- 用户要求每天询问难度
- 用户要求每次发送前验证YouTube视频

### 难度体系
| Level | 特征 |
|-------|------|
| L1 ⭐ | 3-5词超短句 |
| L2 ⭐⭐ | 6-8词简单句 |

### 反馈机制

- 太难了 → 降2级
