# TOOLS.md - 本地环境备忘

Skill 定义工具的用法。这个文件记录**你的环境特有**的配置——设备名、地址、偏好设置等。

## 记什么

- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好声音
- 设备昵称
- 任何环境特有的配置

## 示例

```markdown
### 摄像头
- 客厅 → 主区域，180° 广角
- 大门 → 入口，运动触发

### SSH
- 家庭服务器 → 192.168.1.100, 用户: admin

### TTS
- 偏好声音: "Nova"
- 默认音箱: 厨房 HomePod
```

## 为什么单独放

Skill 是共享的，你的环境是你自己的。分开放意味着你可以更新 skill 而不丢失自己的笔记，也可以分享 skill 而不泄露基础设施信息。
