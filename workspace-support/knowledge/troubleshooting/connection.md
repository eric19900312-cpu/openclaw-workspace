# 故障排查 - 连接问题

## 连接超时

**常见原因：**
- 网络不稳定或 VPN 干扰
- 服务地址/端口配置错误
- 服务器端负载过高
- 防火墙/代理拦截

**排查步骤：**
1. ping 目标地址，检查网络连通性
2. 确认端口号是否正确
3. 更换网络（切换 Wi-Fi/手机热点）测试
4. 检查是否被防火墙拦截
5. 查看服务器状态页面是否在维护

**仍无法解决？** → 提交工单，记录：IP 地址、端口、错误截图、时间点

---

## 认证失败

**常见原因：**
- 密码错误或账户被锁定
- Token/API Key 过期
- 权限不足

**排查步骤：**
1. 确认账户状态正常
2. 尝试重置密码
3. 检查 Token 有效期
4. 确认账户权限配置
