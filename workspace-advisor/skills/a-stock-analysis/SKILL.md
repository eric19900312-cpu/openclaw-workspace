## 📈 分析维度

### 1. 行业赛道（景气度）
- **政策导向**：十四五规划、碳中和、数字经济等
- **行业周期**：上升期/成熟期/衰退期
- **市场需求**：是否具备长期增长逻辑

### 2. 公司基本面
- **盈利能力**：ROE、毛利率、净利率
- **成长性**：营收/利润增速
- **护城河**：品牌、技术、渠道、资源
- **管理层**：诚信、能力、激励机制

### 3. 估值水平
- **PE/PB**：历史分位点
- **股息率**：稳定分红能力
- **DCF**：内在价值估算

### 4. 资金面
- **机构持仓**：公募、私募持股变化
- **北向资金**：外资动向
- **融资融券**：杠杆资金情绪

## 🔧 已安装工具

### 1. a-stock-monitor（A股量化监控）
- **功能**：7维度市场情绪评分、智能选股（短线5策略+中长线7策略）、实时价格监控
- **使用**：
  - 短线选股：`python3 scripts/short_term_selector.py`
  - 中长线选股：`python3 scripts/long_term_selector.py`
  - 市场情绪：`python3 scripts/market_sentiment.py`
- **依赖**：需要安装 `pip3 install akshare flask ccxt`

### 2. stock-prices（股价查询）
- **功能**：查询美股/港股实时价格
- **注意**：主要针对美股（AAPL、NVDA等），A股用 browser 或 akshare

### 3. fundamental-stock-analysis（基本面分析）
- **功能**：基本面评分体系（质量、资产负债表、现金流、估值）
- **使用**：读取 `references/playbook.md` 按步骤分析
