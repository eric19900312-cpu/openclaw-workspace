# 线索磁铁 — 免费转付费转化框架

设计吸引合格潜在客户并将其转化为付费客户的线索磁铁。
来源：Hormozi ($100M Leads), Funnelytics, BusySeed (500+客户), LeadsPicker 研究。
所有输出发送到 `workspace/artifacts/`。

## 使用场景

- 创建免费资源来捕获邮箱/线索
- 设计免费转付费的转化漏斗
- 决定送什么vs收什么
- 构建入门页面或门控内容
- 规划能带来收入的内容营销（不仅仅是流量）

## 不使用场景

- 直接构建付费产品（使用定价心理学技能）
- 写冷 outreach（使用冷 outreach 技能）
- 创建无转化目标的品牌内容
- 不应该门控的 SEO 内容

## 负面例子

- "帮我写博客文章" → 不行。博客文章不是门控线索磁铁，除非是漏斗的一部分。
- "帮我定价服务" → 不行。那是定价心理学。
- "创建社交媒体帖子" → 不行。那是内容营销，不是线索捕获。

## 边缘情况

- ClawHub 上的免费技能 → 可以。那是高级套餐/咨询的线索磁铁。
- SaaS 免费试用 → 可以。试用是内置转化机制的线索磁铁。
- 直播/研讨会 → 可以。现场活动是高转化线索磁铁（平均 20-40% 出席率，2-5% 转化付费）。

## Hormozi 的线索磁铁原则

来自 $100M Leads：线索磁铁为潜在客户**免费解决一个具体、狭窄的问题**，好到让他们想到：*"免费的东西都这么好，付费的得多厉害啊。"*

### 关键规则

你的线索磁铁必须解决多步骤问题的**一个步骤**。它提供快速胜利，但揭示你的付费产品解决的更大需求。

**例子链条：**
- 线索磁铁："5个预约客户的短信模板"（免费，立即有用）
- 揭示："但我还需要自动发送这些..."
- 付费产品：149美元/月的完整短信自动化服务

### 什么让线索磁铁转化

**3个核心原则：**

1. **即时价值** — 必须现在就能用。5分钟或第一封邮件内交付结果。不要30页的PDF感觉像作业。快速胜利建立信任。

2. **信任信号** — 包含证明："500+代理使用过"、推荐信、你的真实姓名/照片、具体结果。没有证明人们会犹豫。

3. **与当前痛点相关** — 匹配他们**现在**挣扎的，不是他们有一天可能需要的。具体的转化；笼统的不转化。

## 7种线索磁铁类型（按2025-26效果排序）

### 1. 模板和即用工具 ⭐⭐⭐⭐⭐
**为什么有效：** 零努力。下载→使用→得到结果。
**转化率：** 所有类型中最高。
**例子：**
- 漏斗模板、邮件模板、电子表格计算器
- 配置文件、代码片段、工作流方案
- 我们的玩法：免费 OpenClaw AGENTS.md + SOUL.md 模板 → 引导到高级 Agent Family Kit

### 2. 互动测验/测评 ⭐⭐⭐⭐⭐
**为什么有效：** 个性化结果+即时满足。用户投入时间回答=更高承诺。
**转化率：** BusySeed 客户用测验替换静态 PDF 后增长 68%。
**例子：**
- "哪种 AI 自动化适合你的业务？"
- "给你的代理安全态势打分"
- "你是什么类型的 outreach？"

### 3. ROI/成本计算器 ⭐⭐⭐⭐
**为什么有效：** 用他们的具体数字量化你的解决方案的价值。
**转化率：** 一家公司归因 45% 的总收入到一个计算器（BusySeed/案例研究）。
**例子：**
- "未接电话让你的业务损失多少？"
- "计算你的 AI 自动化 ROI"

### 4. "不要做"清单 ⭐⭐⭐⭐
**为什么有效：** 模式中断。大多数内容说应该做什么。错误清单创造紧迫感（"我在犯这些错误吗？"）。
**转化率：** 高打开率+分享，因为触发损失厌恶。
**例子：**
- "7个杀死你 OpenClaw 性能的致命错误"
- "5个让你被列入黑名单的冷邮件错误"

### 5. 迷你课程/邮件序列 ⭐⭐⭐
**为什么有效：** 连续几天滴灌价值，建立关系。每封邮件都是一个接触点。
**转化率：** 初始注册较低，但最终购买率较高。
**例子：**
- "5天 Agent 大师课"（每天一课）
- "3天构建你的第一个 n8n 自动化"

### 6. 清单和速查表 ⭐⭐⭐
**为什么有效：** 简单、可操作、易于消费。
**转化率：** 对顶部漏斗很好，但转化付费较低，除非清单揭示缺口。
**例子：**
- "OpenClaw 安全加固清单"
- "发布前自动化清单"

### 7. 直播/演示 ⭐⭐⭐
**为什么有效：** 最高信任建设格式。实时问答。个性销售。
**转化率：** 20-40% 出席率，2-5% 转化付费（LeadsPicker）。
**注意：** 需要安排、直播存在和录制基础设施。

## 线索磁铁设计检查表

发布前验证：

- [ ] 解决**一个问题**（不是"关于X的一切"）
- [ ] 5分钟内交付价值（或第一封邮件）
- [ ] 有明确的下一步 → 你的付费产品
- [ ] 标题是利益驱动的（"得到X"不是"了解X"）
- [ ] 存在信任信号（推荐、用户数、案例研究）
- [ ] 匹配受众认知水平：
  - 不知情 → "不要做"清单（教育问题）
  - 问题感知 → 模板/计算器（部分解决）
  - 方案感知 → 演示/案例研究（证明你的方案有效）
- [ ] 注册摩擦最小（只需姓名+邮箱，不要10个字段的表单）
- [ ] 移动端友好（50%+注册发生在移动端）
- [ ] 感谢页面销售下一步（不要浪费这个页面）

## 免费→付费转化架构

```
线索磁铁（免费、具体、即时价值）
    ↓
注册（捕获邮箱）
    ↓
交付 + 快速胜利（他们使用，得到结果）
    ↓
培育序列（3-5封邮件，更多价值，建立信任）
    ↓
桥梁（揭示你的付费产品解决的更大问题）
    ↓
报价（付费产品/服务，定位为明显下一步）
```

**关键洞察：** 线索磁铁不是销售。是第一次约会。培育序列建立关系。报价是求婚。不要在第一次约会就求婚。

## 关键数据

- 41% 的营销人员说线索生成是他们的 #1 挑战（2025）
- 互动内容（测验、计算器）通常比静态转化更好 — Demand Metric 发现约 2 倍更多线索
- 计算器推动 45% 收入 — 作为异常值可信，但不是典型结果
- 测验替换静态 PDF → 68% 增长注册
- 直播：20-40% 出席率，2-5% 转化付费
- 最优表单字段：2-3 个（姓名+邮箱）。额外字段降低转化 — 估计范围 4-10%+
- 感谢页面是你网站上意图最高的页面 — 用它来销售
