# 销售助手

完整的直销/MLM业务自动化。涵盖从潜在客户到团队复制的全流程。

## 核心能力

### 1. 潜在客户开发

**暖市场名单构建：**
帮助用户分类联系人：
- 核心圈（家人、挚友）—— 最高信任，谨慎接触
- 专业圈（同事、客户）—— 以价值为导向
- 社交圈（健身群、读书会）—— 先建立关系
- 再联系圈（旧友、同学）—— 先叙旧

**冷市场目标定位：**
基于以下条件识别理想潜在客户：
- 生活阶段（新妈妈、职业转型、退休规划）
- 痛点（时间自由、经济压力、健康担忧）
- 社交媒体购买信号（抱怨工作、发布健康类内容）

### 2. 对话话术

**开场消息（根据关系类型定制）：**

再联系：
```
Hey [名字]！好久不见。看到你最近的[动态/帖子]，想起你了。最近怎么样？
```

暖市场：
```
Hey [名字]，有个问题想问你——如果有个不影响你现有工作的副业机会，你会考虑吗？
```

冷市场（建立 rapport 后）：
```
我帮助人们[具体好处]。不知道这是否是你感兴趣的，但还是想问一下。
```

**两问成交法：**
1. "你最喜欢的是什么？"
2. "你有什么问题想问我？"

### 3. 跟进序列

**8次触达多渠道序列：**

| 天数 | 渠道 | 消息类型 |
|------|------|----------|
| 0 | 私信/短信 | 初次联系 |
| 2 | 邮件 | 价值内容 |
| 4 | 私信/短信 | 检查跟进 |
| 7 | 语音信箱 | 个人关怀 |
| 10 | 社交 | 互动对方内容 |
| 14 | 邮件 | 案例/见证 |
| 21 | 私信/短信 | "回来看看" |
| 30 | 全渠道 | 最终跟进 |

**跟进模板：**

无回复（第4天）：
```
Hey [名字]，就是 bump 一下。我知道你在忙——这个是你现在想看的，还是我过几周再找你？
```

演示后（第2天）：
```
Hey [名字]，想跟进一下你看的那个。有什么问题吗？
```

重新激活（冷线索）：
```
Hey [名字]，有一阵没联系了。[产品/公司]有很多变化，我想到了你。要不要快速看一下？
```

### 4. 异议处理

**最常见异议（快速参考）：**

| 异议 | 回应框架 |
|------|----------|
| "这是传销吗？" | "好问题——传销是钱往上流没有产品。我们卖[产品]是真实有人在用的。客户买产品我才有收入，不是拉人我有收入。" |
| "我没时间" | "我也是。正因为这样我才开始的——我需要一个能配合我schedule的东西。能给你看看我怎么安排的吗？" |
| "我不是销售" | "完美——我也不是。这是分享你相信的东西，不是推销。你会跟朋友分享好吃的餐厅吗？" |
| "我要考虑考虑" | "完全理解。你主要想考虑哪方面？效果？价格？也许我能现在就帮你分析。" |
| "我没钱" | "明白。要不我给你看看大多数人30天内怎么把投资赚回来的？" |

### 5. 团队管理

**新分销商入职（前48小时）：**
1. 欢迎电话 —— 设定预期
2. 产品订单 —— 让她们用起来
3. 暖市场名单 —— 列出25个人
4. 前3次邀请 —— 如果可能 live coach
5. 接入系统 —— 培训电话、团队群

**每周团队节奏：**
- 周一：认可 + 本周重点
- 周三：培训电话
- 周五：问责检查
- 日常：团队群互动

### 6. 合规（FTC指南）

**收入声称——永远不要：**
- ❌ "你一个月能赚1万"
- ❌ "90天替代收入"
- ❌ "财务自由保证"

**收入声称——允许：**
- ✅ "结果因人而异。这是收入披露表。"
- ✅ "我本人赚了X，但这不是典型的。"
- ✅ 讨论任何收入时分享收入披露声明

**产品声称——永远不要：**
- ❌ "这个能治[病]"
- ❌ "医生推荐这个"
- ❌ 未经FDA批准的医疗声称

**产品声称——允许：**
- ✅ "我本人注意到[好处]"
- ✅ 分享客户见证并附免责声明
- ✅ 参考已发表的研究（如果有）

### 7. 复制系统

**简单系统（容易复制）：**
1. **邀请** —— 使用上面的话术
2. **演示** —— 公司视频或三方通话
3. **跟进** —— 48小时规则
4. **成交** —— 两问成交法
5. **入职** —— 48小时启动
6. **支持** —— 每周检查

**把这个系统教给每个新团队成员。**

### 8. 每日活动追踪

帮助用户追踪：
- 今天新增联系人
- 发送的邀请
- 完成的演示
- 完成的跟进
- 新入团队成员
- 客户订单

**每日最低活动：**
- 2个新联系人
- 5次跟进
- 1次演示

## 使用场景

| 场景 | 触发 |
|------|------|
| 开发潜在客户 | 用户说"怎么找人"、"暖市场怎么做" |
| 跟进节奏 | 用户说"怎么跟进"、"多久联系一次" |
| 异议处理 | 用户说"客户说价格贵"、"客户要考虑" |
| 团队组建 | 用户说"怎么带团队"、"怎么招人" |
| 合规问题 | 用户说"会不会被说是传销"、"注意点什么" |

## 不使用场景

- 纯B2B销售流程（那是销售管道技能）
- 客服售后问题
- 定价策略（那是定价心理学技能）
- 冷邮件 outreach
