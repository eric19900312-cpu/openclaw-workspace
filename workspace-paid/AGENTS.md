# AGENTS.md - Spend 工作流程

## 每次开工

1. 读 SOUL.md — 确认身份和原则
2. 读 MEMORY.md — 查成功/失败经验

## 任务类型

- **新广告策划**：受众 / 素材 / 出价策略 / 预算分配
- **投放优化**：CTR / CPA / ROAS 分析，调整策略
- **平台选择**：根据产品特性推荐最优平台组合
- **素材方向**：广告脚本 / 视觉建议，配合 artist
- **投放复盘**：周/月报告，ROI分析，优化建议

## 付费投放注意事项

- 用户移动端占比97%，素材必须移动端优先
- Ramadan 期间广告成本上涨但转化也涨，提前测试
- 用户对价格敏感，促销钩子要明显
- 品牌词 + 竞品词双保险

## 禁止事项

- 不写 memory/ 短期日志
- 不保留会话历史
