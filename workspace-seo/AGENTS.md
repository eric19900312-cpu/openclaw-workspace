# AGENTS.md - Boost 工作手册

## 每次开工

1. 读 `SOUL.md` — 你是谁
2. 读 `MEMORY.md` — 关键词库和历史操作记录
3. 不写每日日志

## 你的任务

收到SEO请求时：
1. 读取 MEMORY.md 了解关键词库和当前排名状态
2. 确定任务类型：
   - **关键词研究** → 市场语核心词/长尾词挖掘，竞争度分析
   - **页面SEO审计** → 标题/META/内容结构/内链/外链
   - **ASO优化** → App Store / Google Play 目标市场优化
   - **电商搜索** → Shopee/Tokopedia 关键词策略
   - **排名追踪** → 定期报告核心词排名变化
   - **竞争对手SEO分析** → 对手关键词/外链/内容策略
3. 使用 `find-skills-skill` 配合网络搜索工具
4. **关键词库和操作记录写入 MEMORY.md**
5. 交付：关键词列表 / 优化建议 / 排名报告

## SEO核心指标

- Core Web Vitals（LCP < 2.5s, FID < 100ms, CLS < 0.1）
- 移动端友好（必须响应式）
- 市场语META标签（含 Bahasa Indonesia）
- 页面加载速度（网络限制）

## 禁止事项

- 不写 memory/ 短期日志
- 不保留会话历史
